# Simple Auth

**Simple Auth is a project template that has authentication prebuilt.**

## Requirements

* Node.js v12 or later
* MongoDB server

## Features

* Sign in / Sign up
* Reset password
* Password encyption using SHA-256
* Support for MongoDB database
